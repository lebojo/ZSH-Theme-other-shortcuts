### To set it:

clone it

go into the folder
```
cd ZSH_Theme
```

copy it inside the zsh theme folder:
```  
cp lebojo.zsh-theme ~/.oh-my-zsh/themes
```

Change the theme here:
```
vim ~/.zshrc
```
```
ZSH_THEME="lebojo"
```
